# XHydra-GUI

![Screenshot](xhydra.png)

# Como usar 

$ sudo apt-get install hydra-gtk

$ git clone https://github.com/RIP-Network/XHydra-GUI

$ cd XHydra-GUI

$ bash configure

$ make

$ make install

$ bash xhydra.sh

# Sistemas testeados

* Parrot OS
* Kali Linux
* Ubuntu
